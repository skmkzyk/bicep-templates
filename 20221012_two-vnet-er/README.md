# Change ExpressRoute Gateway SKU

ExpressRoute Gateway の SKU を変えた時に、トラフィックに影響があるかどうかの一例。

# 構成のポイント

としては何もないのですが、`main-before.bicep` が 'ErGw1AZ' でデプロイされ、`main.bicep` は'ErGw2AZ' でデプロイされます。
したがって、まずは `main-before.bicep` でデプロイした後、[Microsoft/Ethr](https://github.com/Microsoft/Ethr) を走らせ、その後 `main.bicep` でデプロイしてみる、という感じです。

Ethr 同士で通信確認するため、サーバ側は `./ethr -s` だけ、クライアント側は `./ethr -c 10.0.0.4 -t pi -d 0` で無限に走らせます。
